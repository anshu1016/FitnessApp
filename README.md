# My Fitness App

This application provides functionalities related to fitness tracking and management. The main components include `Dashboard`, `Exercise`, `Goal`, and `Food`.

## Components Overview

### 1. Dashboard

The dashboard is the central hub for the application. Here, users can get an overview of their fitness activities, recent exercises, goals, and food consumption.

**Features**:
- Display summary of user's activity.
- Quick links to other main components.
- Visualization of weekly progress.
<img width="923" alt="image" src="https://github.com/anshu1016/FitnessApp/assets/69161719/a316e138-0728-45ee-949b-b6ba710ee481">

### 2. Exercise

This component helps users track their physical activities.
<img width="960" alt="image" src="https://github.com/anshu1016/FitnessApp/assets/69161719/b7b49ac3-f03f-44d5-94d1-0a25ed2e22a1">

**Features**:
- Add a new exercise activity: type of exercise, duration, calories burned, etc.
- List all past exercises.
- Edit or delete a particular exercise entry.

### 3. Goal

Setting goals is essential for progress. This component allows users to set, track, and modify their fitness goals.
<img width="960" alt="image" src="https://github.com/anshu1016/FitnessApp/assets/69161719/1e4efb19-9bf1-4ee4-8559-670cf386a25b">

**Features**:
- Set new fitness goals: like running distance, weight loss, etc.
- Track progress towards meeting these goals.
- Update or remove goals.

### 4. Food

A crucial part of fitness is diet. This component assists users in tracking their food consumption.

**Features**:
- Log food items consumed, along with their nutritional values.
- See a history of consumed foods.
- Edit or remove food entries.
<img width="960" alt="image" src="https://github.com/anshu1016/FitnessApp/assets/69161719/f2fc8bcb-3910-4832-8555-451f03b4de49">

## Getting Started

- Clone the repository.
- Install the dependencies using `npm install`.
- Start the application using `npm start`.
